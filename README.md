# 💧 Bebe Água Aí Pai

Estou iniciando na programagramação e criei esse script simples em Python que manda uma notificação a cada 1 hora lembrando você de se hidratar.

## Como funciona

O script roda em segundo plano e envia uma notificação nativa do sistema operacional de hora em hora.

## Requisitos

- Python 3.x
- pip

## Instalação

1. Clone o repositório
git clone https://github.com/Zyrodead/bebe-agua-ai-pai.git
cd bebe-agua-ai-pai

**2. Instale a dependência**
pip install -r requirements.txt

**3. Forma de executar**
py tomaaguaaipai.py

Para encerrar, pressione Ctrl+C no terminal.
